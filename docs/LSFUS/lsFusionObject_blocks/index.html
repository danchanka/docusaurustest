Description

A single OBJECTS block can contain several comma-delimited declarations of object groups. An object group can contain just one object orÂ several ones. In case of a single object, you can use simplified syntax without specifying the name of an object group and using parentheses. The declaration of an object group may be followed by the options of this group. They are listed one after another in an arbitrary order.
