  • EXCLUSIVE operator

  • IF operator

Context independent operators

Context independent property operators differ from the context dependent ones in that they do not just use the upper parameters but define the parameters themselves. As a result, they cannot be part of expressions and can only be used in the = instruction and the JOIN operator:Â 

  • DATA operator

  • FORMULA operator
  • ABSTRACT operator
  • Object group operators
