Starting up a server#

When you create a new project, a server startup configuration is also created by default. You can run it by selecting "Run -> Run 'Run lsFusion server,'" or by clicking on the symbolÂ Â next to the caption Run lsFusion server in the upper right corner. If startup is successful, the last line in the log should be Server has successfully started.

After the server has been successfully started, you can connect to it over a web-based or desktop client via a shortcut on your desktop.
