**+**, **-**, **\***, **/**, **(+)**, **(-)** operators create [properties](Properties.md) responsible for [arithmetic operations](Arithmetic_operators_+_-_....md).

### Syntax

    expression1 + expression2  
    expression1 - expression2  
    expression1 / expression2  
    expression1 * expression2  
    -expression1
    expression1 (+) expression2  
    expression1 (-) expression2  

### Parameters

*expression1, expression2*
